# ISO/IEC 25010 App Testing
The intended technologies to use are:
1. Functional Suitability: Jest (unit & integration) + Cypress (e2e) + Postman (API)
2. Performance Efficiency: k6 (load testing)
3. Compatibility: Docker + GitHub Actions
4. Usability: WCAG
5. Reliability: Postman
6. Security: OWASP ZAP, SonarQube, Dependabot
7. Maintainability: ESLint + SonarQube
8. Portability: Docker + GitHub Actions
